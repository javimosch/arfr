Technology Stack

I use a custom fullstack boilerplate to bootstrap any kind of project.
Is composed by MongoDB, ExpressJs, Angular and NodeJs.

The boilerplate

It uses static generation tools techniques ensuring frontend projects detached from any server side technology.

It is already integrated with a backend boilerplate for APIs creation with helps to reduce development time drastically.

Some features:

  • Frontend Asset bundling (NodeJS, made from the scratch).
  • Frontend Javascript ES6 to ES5 transpiler (Babel)
  • Frontend Static generator capabilities made from the scratch. Damn fast.
  • Frontend Static Templating (HandlebarsJS)
  • Frontend Fast Live sync developement (Firebase, made from the scratch)
  • Frontend Live sync css injection made from the scratch.
  • Frontend CSS Live update from Chrome Dev console (faster developement times!)
  • Frontend I18N workflow
  • Frontend Angular 1 HTML5 routing workflow
  • Frontend developement dosen't crash (Nodemon)
  • Backend Notifications workflow (mailing)
  • Backend mailing templating (100% compatible HTML technique)
  • Backend controllers (faster apis developement)
  • Backend MongoDB + Mongoose + Extra layer
  • Backend Token authentication workflow
  • Backend automatic tasks workflow
  • Backend Shell, File and Database Logging tool with different verbosity levels
  • Backend developement autoreload (Nodemon)
  • and ...

long live to open source ! https://github.com/javimosch/bastack
