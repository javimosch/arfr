Technology Stack

I use a fullstack boilerplate for fast bootstraping.
Is composed by MongoDB, ExpressJs, Angular and NodeJs.

The boilerplate uses static generation tools techniques ensuring frontend projects detached from any server side technology.

It is already integrated with backend capabilities for APIs creation.

Features

  • Frontend Asset bundling.
  • ES6 to ES5 transpiler (Babel / Rollup)
  • Static generation capabilities
  • Client / Server templating (HandlebarsJS)
  • Blazing fast live sync development powered by firebase
  • Angular 1 HTML5 routing workflow
  • Smart developement processes powered by nodemon
  • HTML Mailing powered by mailgun
  • Buil-in controllers / models system on top of mongoose
  • Token authentication workflow
  • Automatic tasks workflow
  • Backend Shell, File and Database Logging tool with different verbosity levels

Its open source and hosted at Github
